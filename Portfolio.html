Physics engine & path tracer

A physics engine and Vulkan path tracer I wrote in C++ from scratch for G10 MYP personal project. Uses GJK along with EPA for collision detection and impulse dynamics to calculate collisions.

A compiler I wrote in C++ for my own custom programming language "newton". Features tokenization pass with lexical analysis, parsing, creating a parse tree, and finally x86_64 assembly compilation for execution.
